linux 文件操作

文件操作也包括文件夹操作（ Linux 系统下万物皆文件）。主要涉及：

  • 文件创建：atom/touch
  • 文件夹创建： mkdir
  • 文件夹删除：rm -r folder
  • 文件移动： mv file ../
  • 文件删除： rm file
